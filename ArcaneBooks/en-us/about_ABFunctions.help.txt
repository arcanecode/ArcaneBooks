
about_ABFunctions

SHORT DESCRIPTION
   This is a listing of the functions available in the ArcaneBooks module.

LONG DESCRIPTION
   Get-ISBNBookData
      Retrieves information for one or more books based on the ISBN number.
      https://github.com/arcanecode/ArcaneBooks/blob/main/Code/Help/Get-ISBNBookData.md

   Get-LCCNBookData
      Retrieves information for one or more books based on the LCCN (Library
      of Congress) number.
      https://github.com/arcanecode/ArcaneBooks/blob/main/Code/Help/Get-LCCNBookData.md

   Open-ABGitHub
      Opens the GitHub website for ArcaneBooks in your default browser.
      https://github.com/arcanecode/ArcaneBooks/blob/main/Code/Help/Open-ABGitHub.md

   Open-AboutArcaneCode
      Opens the ArcaneCode blog to the "About Me" page to learn more about
      the author of ArcaneBooks.
      https://github.com/arcanecode/ArcaneBooks/blob/main/Code/Help/Open-AboutArcaneCode.md
